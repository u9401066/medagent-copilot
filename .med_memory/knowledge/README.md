# Knowledge Base (醫學知識記憶)

> ✅ **此目錄包含通用醫學知識，可跨病人使用**

## 檔案說明

| 檔案 | 用途 |
|-----|------|
| `clinical_knowledge.md` | 臨床決策閾值、用藥劑量 |
| `fhir_functions.md` | FHIR API 使用說明 |
| `task_instructions.md` | 各任務類型處理指南 |
| `task_examples.md` | 任務範例和答案格式 |

## 使用方式

這些檔案會自動被 VS Code Copilot 讀取作為上下文。
你可以在任何任務中參考這些知識。

## 擴充知識

如需新增醫學知識，在此目錄建立新的 `.md` 檔案：
- `medication_dosing.md` - 藥物劑量速查
- `lab_interpretation.md` - 檢驗值判讀
- `clinical_protocols.md` - 臨床處置流程
